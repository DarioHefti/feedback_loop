# Task: Write a Counter Module

Create a simple counter module in TypeScript that:

1. Exports a `createCounter` function that returns a counter object
2. The counter object should have:
   - `increment()` - increases count by 1
   - `decrement()` - decreases count by 1
   - `reset()` - sets count to 0
   - `value` - getter for current count
3. The initial count should be 0

Write the code to `examples/counter.ts`.

## Requirements
- Use TypeScript with proper types
- No external dependencies
- Keep it simple and clean
